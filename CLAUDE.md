## How to use this harness

Before any significant operation (setup, deploy, git, debug), read the relevant skill in `skills/`. Before writing code, read `conventions/`.

## Core Rules

1. Read the skill file before performing any dangerous or irreversible operation.
2. Follow conventions. Do not invent new patterns unless explicitly asked.
3. Add non-obvious discoveries to `memory/lessons.md`.
