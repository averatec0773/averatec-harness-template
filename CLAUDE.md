## About this repo

This repo is a harness — a structured set of files for AI-assisted project management. All files except `README.md` are written for AI agent consumption: optimized for clarity and parseability, not human prose. `README.md` is the only file intended for human readers.

## How to use this harness

Before any significant operation, find and read the relevant skill in `skills/`. Before writing code, read `conventions/`.

## Core Rules

1. Read the skill file before any dangerous or irreversible operation.
2. Follow conventions. Do not invent new patterns unless explicitly asked.
3. Add non-obvious discoveries to `memory/lessons.md`.
