## Rolling rule

When `CHANGELOG.md` exceeds ~10 released versions or ~250 lines, move the oldest version
series into `changelog/0.X.Yx.md`, add a row to the index above, and leave an
"Older releases → `changelog/`" pointer at the bottom of `CHANGELOG.md`. The
[harness](../.claude/skills/harness/SKILL.md) skill enforces this.
